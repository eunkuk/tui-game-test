# tui-game

터미널 기반 던전 탐험 RPG 게임.

## 개요

100층 구조의 **어둠의 탑**을 탐험하는 자동/수동 전투 & 탐험 게임입니다.
영웅을 모집하고 파티를 구성하여 탑의 꼭대기를 정복하세요.

## 주요 기능

- **100층 타워 탐험** — 층마다 자동 생성되는 2D 타일맵, 전투/보물/함정/보스 등장
- **자동/수동 모드** — Tab으로 전환. 자동은 AI 경로탐색, 수동은 WASD/방향키 이동
- **자동 전투** — 영웅 AI가 스킬을 자동 선택 (힐, 스턴, 딜 우선순위)
- **영웅 수집** — 주인공 + 동료 영웅 모집, 레벨업, 스탯 분배
- **장비 시스템** — 무기/방어구/장신구 장착, 상점 구매/판매
- **포션 시스템** — HP/스트레스 회복, 공격/방어/속도 버프 물약
- **아이템 드롭 선택** — 획득 시 장착/보관/사용/버리기 팝업
- **스트레스 시스템** — 스트레스 축적 → 정신이상/미덕 발동
- **횃불 시스템** — 어둠이 깊어지면 기습 확률 증가, 보상 증가
- **세이브/로드** — 자동 저장, 이어하기 지원

## 실행 방법

```bash
npm install
npm start
```

## 조작법

### 던전 탐험
| 키 | 동작 |
|---|---|
| `Tab` | 자동/수동 모드 전환 |
| `↑↓←→` / `WASD` | 수동 이동 |
| `1` `2` `3` | 게임 속도 (1x/2x/3x) |
| `Space` | 일시정지 (자동 모드) |
| `Esc` | 탈출 |

### 아이템 획득 팝업
| 키 | 동작 |
|---|---|
| `1` | 장착 / 사용 (영웅 선택) |
| `2` | 보관 |
| `3` | 버리기 |

### 전투
| 키 | 동작 |
|---|---|
| `1` `2` `3` | 게임 속도 |
| `Space` | 일시정지 |
| `Esc` | 포기 (탈출) |

## 기술 스택

- **TypeScript** + **tsx** (런타임)
- **blessed** (터미널 UI)
- **rot-js** (맵 생성)

## 프로젝트 구조

```
src/
├── App.ts              # 앱 진입점, 화면 전환
├── models/types.ts     # 타입 정의
├── state/              # 상태 관리 (리듀서 패턴)
├── data/               # 영웅, 스킬, 아이템, 몬스터 데이터
├── engine/             # 전투, 루트, AI, 맵 생성 엔진
├── screens/            # 각 화면 (타이틀, 마을, 던전, 전투 등)
└── ui/                 # UI 컴포넌트 (팝업 등)
```
